



# 一、常用命令

## 1. 启动停止MQ

```shell
启动MQ
sudo rabbitmqctl start_app
停止MQ
sudo rabbitmqctl stop
```

**2 启动卸载某个插件**

```shell
# 启动上面列表中的rabbitmq_management ：web管理界面
sudo rabbitmq-plugins enable rabbitmq_management

# 卸载
sudo rabbitmq-plugins disable rabbitmq_management
```
